# Task App

This is a React application built using Next.js, TypeScript, Tailwind CSS, and Redux Toolkit. It includes various UI components, form handling with Formik and Yup, and integration with external libraries like Axios for API requests and Radix UI for accessible and customizable UI components.

## Features

- Next.js Framework: Enables server-side rendering, static site generation, and optimized page-based routing.

- TypeScript: Provides static typing to catch errors during development.

- Tailwind CSS: A utility-first CSS framework for creating custom and responsive designs quickly.

- Redux Toolkit: Manages application state efficiently.


- Embla Carousel: Adds carousel/slider functionality.
- Radix UI Components: Prebuilt and accessible components like Accordion, Avatar, Label, and Icons.

- Lucide Icons: A collection of open-source icons.
Axios: For handling HTTP requests.


## Getting Started

To set up and run the project locally, follow these steps:

### Prerequisites

-  Node.js (version 14 or higher recommended)
-  npm or yarn

### Installation

1. Clone the repository:

```javascript
git clone https://github.com/codefast-in/AnglaraDSNextJSTask.git
cd AnglaraDSNextJSTask
```

2. Install dependencies:

```js
npm install
# or if you prefer yarn
yarn install
```

### Available Scripts

In the project directory, you can run the following commands:

- *npm run dev*: Runs the app in development mode. Open http://localhost:3000 to view it in the browser.

- *npm run build*: Builds the app for production to the .next folder.

- *npm run start*: Starts a production server for the app.

- *npm run lint*: Lints the code using ESLint.

### Dependencies

- Core Libraries: react, react-dom, next, typescript
- UI Libraries: shadcn, lucide-react, tailwindcss, tailwindcss-animate
- State Management: @reduxjs/toolkit, react-redux
- HTTP Requests: axios
- Utilities: clsx, tailwind-merge, class-variance-authority
